Conceptos de Regresión Logística

La regresión logística es un algoritmo de aprendizaje supervisado usado para problemas de clasificación binaria. Su objetivo es modelar la probabilidad de que ocurra un evento (ej. cancelación de una reserva).

Función sigmoide

La regresión logística utiliza la función sigmoide para transformar valores reales en probabilidades entre 0 y 1: 

σ(z) = 1 / (1 + e-z)

Métricas principales

  • Exactitud (accuracy): porcentaje de aciertos sobre el total.
  • Precisión: proporción de verdaderos positivos entre los predichos como positivos.
  • Recall (sensibilidad): proporción de verdaderos positivos correctamente detectados.
  • F1-score: media armónica entre precisión y recall.

Práctica en esta aplicación

Aquí trabajamos con un dataset de reservas de hotel. Las variables de entrada incluyen:

  • dias_antelacion
  • tipo_habitacion (categórica)
  • estancia_previa
  • tamano_grupo

El modelo predice si la reserva se cancela (Sí) o no (No).
