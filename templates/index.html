Introducción

Este proyecto académico tiene como propósito investigar y presentar casos de uso relevantes del Machine Learning Supervisado aplicados en diferentes industrias. La actividad busca que los estudiantes comprendan cómo los algoritmos supervisados permiten resolver problemas reales mediante el análisis de datos etiquetados, generando soluciones que aportan valor a organizaciones y usuarios.

El desarrollo se implementa en una aplicación web utilizando Flask como framework backend y Jinja2 para la visualización, junto con estilos en CSS. De esta manera, se integran tanto los aspectos técnicos de programación como la investigación documental.

Objetivo de la actividad

Desarrollar una aplicación en Flask que muestre cuatro casos de uso del aprendizaje supervisado en distintos sectores, describiendo para cada uno:

  • El problema que resuelve.
  • Los algoritmos aplicados.
  • Los beneficios alcanzados.
  • Ejemplos concretos de empresas o proyectos.
