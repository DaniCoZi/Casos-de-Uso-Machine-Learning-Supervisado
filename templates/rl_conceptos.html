Conceptos básicos · Regresión lineal

En este proyecto modelamos el ingreso de un negocio como función de la inversión en publicidad y el precio de venta, utilizando regresión lineal múltiple. El objetivo es entender el efecto de cada variable y predecir el ingreso para nuevos valores de publicidad y precio.

Regresión lineal

Se dice que existe regresión de los valores de una variable con respecto a los de la otra cuando hay alguna línea de regresión que se ajusta, de forma más o menos clara, a los valores observados. Se usa para la identificación de relaciones potencialmente causales o, cuando la relación causal está clara, para predecir una variable a partir de otra.

En fenómenos biológicos la relación entre variables suele ser aleatoria, pues múltiples factores (a menudo desconocidos) influyen en la relación. En escenarios deterministas, las mediciones caen muy próximas a la línea de regresión; la cercanía de los puntos a dicha línea está determinada por el error de medición.

Definición de regresión lineal

La regresión lineal es una técnica de modelado estadístico que examina la relación entre una variable dependiente y una o más variables independientes. Su objetivo es encontrar la mejor recta (o hiperplano) que se ajusta a los datos, permitiendo predecir la variable dependiente \(Y\) a partir de las independientes \(X\).

En su forma simple:

{% raw %} 
      \[ Y = \beta_0 + \beta_1 X + \varepsilon \]
  • Y: variable dependiente.
  • X: variable independiente.
  • \(\beta_0\): intercepto (valor de Y cuando X=0).
  • \(\beta_1\): pendiente (cambio en Y por unidad de cambio en X).
  • \(\varepsilon\): término de error (diferencia entre observado y predicho).

En nuestro caso (múltiple):

      \[ \hat{Ingreso} = \beta_0 + \beta_1 \cdot Publicidad + \beta_2 \cdot Precio + \varepsilon \]
{% endraw %}

Supuestos del modelo clásico

Linealidad

Supuesto fundamental: la relación entre la variable dependiente y las independientes es aproximadamente lineal. Si no lo es, el modelo lineal puede ser inadecuado y producir predicciones sesgadas o poco precisas.

Homoscedasticidad

La varianza de los errores es constante a lo largo del rango de predicción. Si la dispersión de los residuos cambia con el nivel de X (heterocedasticidad), se afectan la eficiencia de los estimadores y la validez de pruebas de significancia.

Normalidad del error

Los residuos siguen (aprox.) una distribución normal con media cero. Este supuesto es relevante para la inferencia (p-valores, intervalos de confianza); para predecir medias, puede relajarse con tamaños muestrales grandes (TCL).

Independencia de los errores

Los residuos no deben estar correlacionados entre sí. En series de tiempo, la autocorrelación viola este supuesto y requiere correcciones (por ejemplo, incorporar rezagos o usar modelos específicos para datos temporales).

Flujo del ejercicio

  1. Carga de datos: dataset con Publicidad, Precio e Ingreso.
  2. Entrenamiento: ajuste por mínimos cuadrados (OLS).
  3. Predicción: ingresar nueva tupla (Publicidad, Precio) para obtener \(\hat{Ingreso}\).
  4. Visualización: proyección 2D (Ingreso vs Publicidad con Precio fijado en la mediana).
